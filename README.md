# Desperate Measures

Critical Role-inspired Desperate Measures module for Foundry VTT v13 and D&amp;D5e.

## Version 0.4.20

The module automatically creates one player-visible Journal Entry with a single rules-reference page. The page follows the module language selected by the primary active GM and includes:

- the current Bloodied threshold, 0 HP, active-combat, long-rest reset and chat-report settings;
- the failed death-save cost and use sequence;
- all six Desperate Measures and their timing restrictions;
- roll history, maximum-damage timing, spell-slot recovery and manual-resolution notes.

The module flags the Journal and its page, preventing duplicates after a restart. Its managed page refreshes when relevant settings or the module language change.
